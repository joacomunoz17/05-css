PROPIEDADES DE CSS

Background-color

es un propiedad de CSS que define el color de fondo de un elemento. ejemplo

Background-image

establece una o más imágenes de fondo para un elemento. ejemplo

Color

selecciona el valor de color de primer plano del contenido de elemento de texto y decoraciones de texto. ejemplo

Text-align

El text-align sirve para alinear textos (izquierda, derecha y centro). ejemplo

Width

Especifica la anchura del area de contenido de un elemento.

height

especifica la altura de un elemento.

border

permite definir en una única regla todos los bordes de los elementos seleccionados.Se puede utilizar border para definir el o los valores siguientes: border-width, border-style, border-color.

ejemplo

border-collapse

se utiliza para fusionar los bordes. Ésto tiene una gran influencia sobre la presentación y el estilo de las celdas de tabla. ejemplo

Display

establece los tipos de visualizacion interna y externa de un elemento. ejemplo

justify content

define como el navegador distribuye el espacio entre y alrededor de los item flex, a lo largo de su contenedor. ejemplo
